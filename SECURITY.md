# Security Policy

## Supported Versions

DigBick is still in early development.

Security updates are provided for the latest version of the project only.

| Version                        | Supported |
| ------------------------------ | --------- |
| Latest release / `main` branch | ✅         |
| Older versions                 | ❌         |

## Reporting a Vulnerability

If you discover a security vulnerability in DigBick, please report it privately instead of opening a public GitHub issue.

Contact:

`satriyobudiu@gmail.com`

Please include as much detail as possible:

* Description of the vulnerability
* Steps to reproduce
* Potential impact
* Affected version or commit
* Any suggested fix, if available

I will try to respond as soon as possible.

If the vulnerability is confirmed, I will work on a fix and publish an update. Public disclosure should wait until a fix is available.

Please do not use security reports for general bugs, feature requests, or support questions.
